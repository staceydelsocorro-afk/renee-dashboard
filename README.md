# Renée Marino — Live Social Media Dashboard

Interactive, editable dashboard tracking social media performance across Instagram, YouTube, Facebook, LinkedIn, and Twitter/X — including competitive analysis vs. Vinh Giang and Vanessa Edwards, a viral pattern review, traffic-light status, weekly action plan, and a 90-day calendar pacing Stacey's content production against Renée's delivery.

**View it live →** _(GitHub Pages URL will go here once Pages is enabled in repo settings)_

## What's editable

Every piece of text — section headings, KPI numbers, competitor stats, traffic-light bullets, action items, phase descriptions, role cards, calendar entries, even the footer. Click any text and rewrite it. Edits save automatically in the viewer's browser.

The 90-day calendar supports drag-and-drop between days, add (+) and remove (×) on each item, and an R/S badge to switch ownership between Renée and Stacey.

## Files

- `index.html` — the dashboard (this is what GitHub Pages serves)
- `renee_dashboard.html` — identical copy for download / offline use

## Updating

To refresh metrics each week, edit `index.html` directly in GitHub (pencil icon on the file page), commit, and GitHub Pages will re-deploy in ~30 seconds.
